## 1.0.0

- Initial release.
- Core reactive engine: Signal, Computed, Effect, batch, ReactiveScheduler
- ReactiveList with reactive transforms and UltraFor integration
- Reactive widgets: XReactive, UltraBuilder, UltraObserver, XSuspense, XTransition
- X Helper API: X.text, X.show, X.opacity, X.container, X.button, X.transition
- Async resource: CapsaResource with loading/ready/error states
- MVVM architecture: ScreenModel, Business, Repository, Datasource
- DI container: UltraDI with scopes, singletons, factories, async support
- Low-level render object widgets: UltraReactiveBox, UltraReactiveFlex, UltraReactiveStack, ReactiveText
- Reactive grid/wrap/list layouts: UltraGrid, UltraWrap, UltraCollection, UltraFor
- Code generator: @Capsa annotation with feature scaffolding
- CapsaLogger with categories, levels, and ring buffer history
- ReactiveDevTools for graph inspection and leak detection
- Router integration with ReactiveRouteObserver
- 9 example profiles demonstrate all major APIs in a real CRUD app
